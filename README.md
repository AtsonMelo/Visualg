# Visualg

Este foi meu primeiro repositório de programação.

Aqui comecei meus estudos de lógica usando Visualg, praticando conceitos como variáveis, entrada e saída de dados, operadores, estruturas condicionais, laços de repetição, vetores, matrizes e construção de algoritmos.

Hoje sigo evoluindo para C#, .NET, Windows Forms, PowerShell, automação, eletrônica aplicada e diagnóstico de hardware, mas este repositório representa o início da base lógica que sustenta meus projetos atuais.

## Objetivo do repositório

Registrar minha primeira fase de aprendizado em programação e organizar os exercícios feitos em Visualg.

## Conceitos estudados

- Algoritmos
- Variáveis
- Tipos de dados
- Entrada e saída
- Operadores aritméticos
- Operadores lógicos
- Condicionais
- Laços de repetição
- Vetores
- Matrizes
- Procedimentos e funções

## Organização

- `algoritmos/`: exercícios organizados por tema.
- `docs/`: anotações e diário de aprendizado.
- `legado/`: arquivos originais ou materiais antigos preservados.
