# Conceitos estudados

## Algoritmo

Um algoritmo é uma sequência organizada de passos para resolver um problema.

## Variável

Uma variável é um espaço usado para armazenar um valor que pode mudar durante a execução do programa.

## Entrada

Entrada é o momento em que o programa recebe dados do usuário.

## Processamento

Processamento é a parte em que o programa calcula, compara ou transforma os dados.

## Saída

Saída é o resultado mostrado ao usuário.

## Condicional

Uma estrutura condicional permite que o algoritmo tome decisões.

Exemplo:

    se idade >= 18 entao
       escreva("Maior de idade")
    senao
       escreva("Menor de idade")
    fimse

## Laço de repetição

Um laço de repetição executa o mesmo bloco de código várias vezes.

Exemplo:

    para contador de 1 ate 10 faca
       escreval(contador)
    fimpara
